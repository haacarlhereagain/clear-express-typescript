## install dependencies
```bash
yarn install / npm install
```
## set .env-file
```bash
set .env.PORT
```
## dev
```bash
dev: yarn serve / npm run serve
```
## build
```bash
build: yarn build / npm run build (result in /dist/index.js)
```
## launch a built project
```bash
launch a builded project: node ./dist/index.js
```
